He estado utilizando DigitalOcean desde el 2012 y quiero compartir como es que configuro un droplet (VPS) cada que necesito uno para un proyecto o simplemente para probar nuevas cosas.

Una vez dentro de nuestra cuenta de DigitalOcean en la barra lateral izquierda seleccionamos el menu de Droplets despues dar click en el boton Create y nos desplegara un submenu donde daremos nuevamente click en la opcion de Droplets.

Sistema Operativo

El paso anterior nos llevara a la siguiente pagina donde configuraremos nuestro Droplet y como primer paso nos pedira que seleccionemos el sistema operativo que tendra nuestro servidor, por default viene seleccionado Ubuntu que es el que regularmente uso.

Capacidad

Ahora debemos seleccionar cuanta memoria ram, espacio y procesador queremos para nuestro Droplet y generalmente el mas economico sera mas que suficiente.

Backups

Despues nos da la opcion de agregar Backups para con esto contar con un respaldo de nuestro servidor por si llega a haber un problema poder crear otro servidor con el respaldo.

Region

Ahora debemos seleccionar una region donde nuestro Droplet sera creado.

Opciones adicionales

En las opciones adicionales podemos seleccionar si queremos Private networking que nos sirve por si tenemos mas de un Droplet para que estos se puedan comunicar sin que nos cobren por ese trafico, ademas que hace la comunicacion mas rapida entre servidores. Tambien podemos seleccionar si tenemos llaves SSH dadas de alta, si no, entonces nos enviaran un correo con la informacion del Droplet para ingresar con un usuario y password. Al final de estas opciones nos pregunta cuantos Droplets queremos crear y cual es el nombre que le queremos dar al servidor.

Crear droplet

Por ultimo le damos click al boton Create y el proceso de creacion iniciara.

Finalizar

Este proceso de creacion dura poco menos de 30 segundos y despues de terminar podremos ingresar a nuestro Droplet para poder configurarlo.

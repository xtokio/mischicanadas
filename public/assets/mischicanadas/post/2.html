LEMP Stack es cuando configuramos nuestro servidor e instalamos Linux NGINX MySQL PHP esa sera la base de nuestro servidor para usarlo en nuestros desarrollos web.

Lo primero que debemos hacer es actualizar nuestro sistema operativo.

Instalamos NGINX

Habilitamos NGINX en el firewall

Instalamos MySQL

Instalamos PHP

Con esto nuestro servidor queda configurado con NGINX , MySQL y PHP si nos dirigimos a la IP del servidor deberiamos de ver la pagina de bienvenida de NGINX.
